O sistema de loja desenvolvido em Python, utilizando SQLite como banco de dados e Pandas para tratamento e análise de dados.
O projeto simula o funcionamento de uma loja de tecnologia, com cadastro de clientes, produtos, pedidos e um painel administrativo.
Este projeto foi desenvolvido com o objetivo de aprimorar o aprendizado nas principais tecnologias utilizadas, como Python, SQLite e Pandas.
Por ainda estar em fase de aprendizado, nada melhor do que a construção de um CRUD (Create, Read, Update e Delete) para treinar conceitos fundamentais de programação, banco de dados relacional e manipulação de dados.
